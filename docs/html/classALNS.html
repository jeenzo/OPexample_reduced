Detailed Description

ALNS class to act on a specific PI with index of operators to be used.

Assumes that PI is already initialized, with batches assigned to workers, and their related routes and fatigue computed (computeTotalPerformance() already called on feasible initial solution).

Best and current solutions are saved as workers. New solution is the current PI workers
